1. Virtual DOM

Virtual DOM 是 一种用JS对象描述DOM对象的方式。修改VirtualDOM的时候，只需要改变原来的DOM对象的属性，以提高重新渲染的性能。

3.React生命周期函数

定义：某一个时刻组件自动执行的函数。（生命的进程里总会发生的事情）

例子：

render函数在state或者props改变的时候自动调用。

几个重要的节点：

  1. 初始化（Initialization）
  2. 挂载（Mounting）
  3. 更新（Updation）
  4. 去挂载（Unmounting）
